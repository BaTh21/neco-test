Frontend

Navigate to frontend folder:

cd frontend/react

Install dependencies:

npm install

Run development server:

npm run dev

http://localhost:5173

Backend

Navigate to backend folder:

cd backend

Install dependencies

pip install -r requirements.txt

Run development server:

python -m uvicorn app.main:app --reload

Docker 

To build new docker image using docker compose

docker compose up --build

To Re-build docker image using docker compose

docker compose up -V
